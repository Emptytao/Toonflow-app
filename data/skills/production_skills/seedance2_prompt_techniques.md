# Seedance 2.0 提示词增强技法

## 使用边界

- 本技法只作为 Seedance 2.0 相关任务的可选增强参考。
- 不改变生产 Agent 阶段 5 的写入模式；Seedance 2.0 仍走 `纯文本多参模式`。
- 不新增分镜面板字段，不要求输出新的 XML。
- 本项目统一使用 `@参考N` 语法；不要改成 `@图片N` / `@视频N` / `@音频N`。
- BGM / 配乐只可作为旁路推荐建议，不写入视频生成主 prompt。

## Seedance 2.0 输入约束

| 输入类型 | 数量上限 | 支持格式 | 大小限制 |
|---|---:|---|---|
| 图片 | 9 张以内 | jpeg / png / webp / bmp / tiff / gif | 每张小于 30 MB |
| 视频 | 3 个以内 | mp4 / mov | 每个小于 50 MB，总时长 2-15 秒 |
| 音频 | 3 个以内 | mp3 / wav | 每个小于 15 MB，总时长不超过 15 秒 |
| 总文件数 | 12 个以内 | - | 图片、视频、音频合计 |

生成时长建议控制在 4-15 秒。4-5 秒内不要塞入过多场景、转场或复杂动作链。

## 参考用途标注

在 `参考定义` 中，每个 `@参考N` 都必须说明清楚用途。

可用用途包括：

- 角色形象参考
- 场景 / 背景参考
- 道具 / 产品外观参考
- 首帧参考
- 尾帧参考
- 运镜参考
- 动作编排参考
- 特效 / 转场参考
- 节奏 / 节拍参考
- 声音 / 音色参考
- 音效参考

不要只写“参考 @参考N”。必须说明参考的是形象、场景、动作、运镜、节奏、声音还是特效。

## 提示词结构

Seedance 2.0 的主 prompt 应优先按以下顺序组织：

1. 主体 / 人物设定
2. 场景 / 环境
3. 动作 / 运动描述
4. 运镜语言
5. 分时段或分画面推进
6. 转场 / 特效
7. 台词 / 口型 / 音色
8. 环境音 / 动作音 / 拟音
9. 风格 / 氛围

当单个 `track` 接近 10 秒或更长时，优先使用分画面或分时段表达，让动作、镜头、台词和音效在时间上连续推进。

## 运镜语言

常用运镜：

- 推镜头 / 慢推：镜头向主体靠近
- 拉镜头 / 后拉：镜头远离主体
- 左摇 / 右摇：镜头水平旋转
- 上摇 / 下摇：镜头垂直旋转
- 跟随镜头 / 跟拍：镜头跟随主体移动
- 环绕镜头：镜头围绕主体旋转
- 一镜到底：全程无剪辑的连续镜头

高级运镜：

- 希区柯克变焦：推拉结合变焦，制造眩晕或心理压迫感
- 鱼眼镜头：超广角畸变视角
- 低角度仰拍：强化主体压迫感或英雄感
- 俯拍 / 鸟瞰：从高处向下建立空间关系
- 第一人称主观视角：从角色视角推进动作
- 快速摇镜：极速水平旋转并产生运动模糊
- 机械臂跟随：多角度灵活跟随人物视线或动作路径

景别：

- 极致特写：眼睛、嘴巴、手部等关键细节
- 面部特写：人脸或表情充满画面
- 中近景：头部与肩部
- 中景：腰部以上
- 全景：完整人物与动作
- 远景 / 建立镜头：完整环境与空间关系

## 台词、口型与音色

- 普通对白必须写清角色正在说话，嘴部开合说话。
- 内心独白必须标注 `内心OS`，嘴部紧闭不动。
- 画外音必须标注 `画外音VO`，嘴部紧闭不动或人物不在画面中。
- 有参考音频时，音色写为“音色：参考定义中的对应音频”。
- 无参考音频时，可按 9 维度生成音色：性别、年龄音色、音调、音色质感、声音厚度、发音方式、气息、语速、特殊质感。
- 无台词时，不写“说：”，只标注“无台词”。

## 音效与 BGM 边界

- 视频生成主 prompt 只写环境音、动作音、拟音。
- 音效必须落到具体画面或动作上，例如脚步声、风声、衣料摩擦声、门轴声、金属碰撞声。
- 不在视频生成主 prompt 中写 BGM、配乐、旋律、乐器风格、音乐卡点。
- 如需 BGM，只生成独立“BGM 推荐建议”，供人工参考，不参与视频生成。

## 常见错误校验

生成或审核 Seedance 2.0 提示词时，重点检查：

- 引用模糊：每个 `@参考N` 是否说明用途。
- 指令冲突：是否同时要求固定镜头与环绕镜头、静止镜头与快速跟拍等互斥动作。
- 内容过载：4-5 秒内是否塞入过多场景或转场。
- 素材无归属：上传素材是否都在 `参考定义` 中说明用途。
- 时长不匹配：动作复杂度是否适配 4-15 秒段落。
- 台词遗漏：普通对白、内心OS、画外音VO 是否完整保留。
- 口型错误：OS / VO 是否明确嘴部紧闭或人物不在画面中。
- 音效漂浮：音效是否落在具体画面或动作上。
- BGM 泄漏：主视频 prompt 是否误写了 BGM / 配乐 / 旋律 / 乐器风格。
